# AirFuture Branding & Generation Rules

このガイドラインは、AirFutureブランドの視覚的整合性を保ち、正規製品のイメージを正しく伝えるために、AIが画像や動画を生成する際に必ず守るべきルールです。

## 1. 最優先使用素材（正規製品イメージ）
- **正規製品のマスター画像**: `/Users/akihironishi/af-mini-specialLP/dist/favicon.png`
- **ルール**: 
    - 画像生成（`generate_image`）や動画生成を行う際は、必ず上記のパスにあるファイルを `ImagePaths` に含めるか、これをプライマリなリファレンスとして使用すること。
    - 製品のフォルム、細部のディテール（メッシュ、ロゴの位置、ボタン形状）は、この素材のデザインから逸脱してはならない。

## 2. 推奨アセットの使用
- **アセットディレクトリ**: `/Users/akihironishi/af-mini-specialLP/public/images`
- **ルール**:
    - 背景素材、補助的なアイコン、周辺機器等のイメージを採用する際は、可能な限りこのディレクトリ内にある既存素材から採用すること。
    - 新規に生成する場合も、このディレクトリ内の素材が持つトーン＆マナー（清潔感、高級感、パステル/白を基調とした色使い）を継承すること。

## 3. クオリティ・スタンダード
- **トーン**: ハイブランド、ハイクオリティ、清潔感。
- **テキスト**: 指定がない限り、表示はすべて日本語で行うこと。
- **配置**: 黄金比を意識し、製品が主役（Hero）となるマージンを十分に確保すること。

## 4. 実行手順
1. 生成ツール（`generate_image`等）を呼び出す前に、必ず上記ディレクトリの内容を確認（`list_dir`）し、最適な素材を特定する。
2. プロンプト内で「正規製品素材（favicon.png）のデザインを維持すること」を明示的に指定する。
